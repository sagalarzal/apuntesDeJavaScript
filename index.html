Apuntes de JavaScript

    Las funciones declarativas, podemos mandar llamar la función antes de que ésta sea declarada, y con la expresión de función, no, tendríamos que declararla primero, y después mandarla llamar

    Las var definidas dentro de las funciones no la puedo usar por fuera. Las var cambiadas dentro de la funcion, cambian por fuera. Si las vuelvo a declarar, quedan dentro del scope interno y externo separadas cada una es diferente

    Hoisting: En Js todas las variables var se declaran al inicio como undefined y luego se inicializab eb el renglon que las escribimos. Conclusión: declara las funciones y variables globales al inicio

    Coerción: cambiar el tipo del valor (variable). Implicita (4 + "7" = 47 es concatenación, y da "47" como string. pero 4 * "7" = 28 si da numero) y explícita (c = String(a); convierte mi numero a en string "41").

    Truthy: valores que por defecto son verdaderos ej: 2, a, " ", -1, funciones().

    Falsy: valores que por defecto son falsos ej: NaN, undefined, null.

    Operadores: +, -, *, /, !, =, ==, ===, <, >, <=, >=, &&, ||, ?

    Operador ternario ? ej: condition ? true : false;

    todos los elementos del array deben ser del mismo tipo

    find() : Devuelve el primer elemento del array que cumpla con la condición dada
    foreach() : Ejecuta lo que le definamos una vez por cada elemento de nuestro array
    some() : Comprueba si al menos un elemento del array cumple con la condición que le damos
    filter() : Devuelve todos los elementos del array que cumplan con la condición dada
